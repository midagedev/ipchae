# IPCHAE Service (Svelte + Supabase)

SvelteKit 기반 통합 서비스 워크스페이스입니다.  
이 디렉토리는 Editor + Starter/Parts + Share/Collab + Gamification 트랙을 병렬 오케스트레이션으로 개발합니다.

## Quick Start
```sh
cd /Users/hckim/repo/ipchae/ipchae-service
npm install
npm run dev
```

## Main Scripts
```sh
npm run check
npm run test
npm run build
```
