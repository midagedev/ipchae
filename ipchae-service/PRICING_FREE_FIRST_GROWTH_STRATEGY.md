# IPCHAE Pricing Strategy (Free-first, Growth-first)

기준일: 2026-03-01  
대상: iOS/iPadOS 우선 출시, Android 후행

## 1. 목표
1. 사용자 증가(활성 사용자/작품 생성/공유 전환)를 최우선으로 한다.
2. 무료 사용자도 제품의 핵심 가치(만들기/저장/공유)를 충분히 경험하게 한다.
3. 과금은 "제한 해제"보다 "추가 가치"에 집중해 반감을 줄인다.

## 2. 가격 정책 원칙
1. **No Ads**: 창작 도구 특성상 광고는 이탈을 크게 유발하므로 기본 비채택.
2. **핵심 제작 루프는 무료**: Draw -> Build -> Polish -> Export(기본 포맷)는 무료 유지.
3. **과금 포인트는 파워 유즈케이스**: 협업/팀관리/고급 포맷/대용량/고급 히스토리.
4. **프로젝트 수 하드캡 최소화**: 너무 낮은 프로젝트 제한은 초기 리텐션을 훼손.
5. **명확한 Free Promise**: 무료에서 가능한 것/안 되는 것을 단순하고 투명하게 고지.

## 3. 시장 벤치마크 (요약)
1. Procreate: 일회성 구매(구독 없음)로 강한 호감/입문 장벽 낮음.
2. Nomad Sculpt: 저가 일회성 + IAP 구조.
3. Shapr3D: Freemium, 무료에서도 모델링 가능하나 프로젝트/고급 익스포트에서 차등.
4. Figma: 무료 Starter를 유지하면서 팀/고급 협업에서 유료 전환.
5. Blender: 완전 무료로 생태계 확장, 수익은 다른 채널(후원/서비스)로 보완.

해석:
1. 창작 툴은 무료/저가 진입장벽이 성장에 매우 유리하다.
2. 유료 전환은 "전문 생산성"에서 발생하는 경향이 크다.

## 4. 제안 플랜 (v1)
## 4.1 Free (기본)
1. 대상: 학생/입문자/취미 사용자
2. 포함:
   1. 핵심 3D 편집 기능 전체
   2. 기본 Export (STL/PLY)
   3. 로컬 프로젝트 무제한
   4. 클라우드 동기화 기본 용량 (예: 2GB)
   5. 공유 링크 생성(월 제한 완만하게, 예: 30회)
3. 제한:
   1. 팀 협업 좌석/권한 관리 없음
   2. 고급 Export 포맷 없음
   3. 장기 버전 히스토리 기간 제한 (예: 30일)

## 4.2 Plus (개인 파워유저)
1. 가안 가격: 월 $6~8 / 연 $60~80
2. 포함:
   1. 고급 Export 포맷(OBJ/GLB/USDZ 등)
   2. 클라우드 용량 확대 (예: 100GB)
   3. 버전 히스토리 확장 (예: 180일)
   4. 공유/리믹스 제한 상향 또는 사실상 무제한
   5. 우선 처리 큐(동기화/익스포트)

## 4.3 Team (스튜디오/클래스)
1. 가안 가격: 에디터 좌석당 월 $12~18
2. 포함:
   1. 팀 프로젝트/공동 편집/역할 기반 권한
   2. 감사 로그/소유권 관리
   3. 관리자 콘솔, 일괄 과금
   4. 교육용/학급 할인 정책(별도)

## 5. 무엇을 과금하면 안 되는가
1. 브러시/조형의 기본 도구 자체 잠금
2. 첫 Export 자체 차단
3. 지나치게 낮은 프로젝트 수 제한(예: 1~2개)
4. 강제 광고 노출

## 6. 전환 장치 (유저친화적)
1. "막기" 대신 "확장" 메시지 사용: 기능 차단 문구 대신 상위 플랜 가치 설명.
2. 자연스러운 트리거:
   1. 고급 포맷 export 시
   2. 팀 초대/권한 설정 시
   3. 히스토리 복원 기간 초과 시
3. 7일 Plus 체험: 첫 고급 기능 사용 순간 즉시 활성화.
4. 학생/교사 무료 또는 큰 폭 할인: 초기 네트워크 확산에 유리.

## 7. 실험 로드맵 (출시 후 12주)
1. 실험 A: Free 공유 제한(30/60/무제한) -> 공유율/리텐션 비교
2. 실험 B: Plus 가격점($5.99 vs $7.99) -> 전환율/환불율 비교
3. 실험 C: 히스토리 보존 기간(30일 vs 90일) -> 결제 전환 영향
4. 실험 D: 교육 할인 진입 배너 위치 -> 활성 사용자 증가율

## 8. KPI (가격 정책 성공 기준)
1. 북극성: 주간 활성 창작자(WAC)
2. 핵심:
   1. D1/D7/D30 리텐션
   2. 첫 작품 완성률
   3. 공유 전환율
   4. 무료->유료 전환율
   5. 유료 해지율(월)
3. 가드레일:
   1. 과금 노출 후 세션 이탈률 급증 금지
   2. 무료 사용자 첫 3회 세션 성공률 유지

## 9. 출시 권장안
1. 초기 3개월은 **Free 매우 관대 + Plus 단일 플랜**으로 단순 시작
2. Team 플랜은 협업 수요 지표가 확인되면 2차 오픈
3. 광고 모델은 최소 1년 보류
