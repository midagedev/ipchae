# UI Localization + Icon Policy

기준일: 2026-03-01  
대상: `/Users/hckim/repo/ipchae/ipchae-service`

## 목적
초등 고학년/중학생 중심 UX에서 `쉬운 인지`와 `도구의 전문성`을 동시에 유지하기 위해, 아이콘/다국어 노출 규칙을 고정한다.

## 대상 사용자 톤
1. 초등 고학년/중학생(2D 드로잉 앱 경험자)
2. 지나치게 유아적인 표현은 피하고, 짧고 명확한 제작 도구 톤을 유지한다.

## 언어 정책
1. 지원 언어: `한국어(ko)`, `English(en)`, `日本語(ja)`
2. Locale 토글은 즉시 반영되어야 하며, 새로고침 후에도 유지되어야 한다.
3. 번역 우선순위: `ko -> en -> ja`
4. 미번역 시 fallback은 한국어다.

## 아이콘 정책
1. 주요 메뉴/CTA는 `아이콘 + 텍스트`를 기본으로 한다.
2. 아이콘 단독 버튼은 기능이 매우 명확한 경우(예: 확대/축소)만 허용한다.
3. beginner 모드에서는 텍스트 길이를 짧게 유지한다.

## 용어 표준(핵심)
1. Draw: `그리기 / Draw / 描く`
2. Fill: `채우기 / Fill / 塗る`
3. Erase: `지우기 / Erase / 消す`
4. Undo: `실행취소 / Undo / 元に戻す`
5. Redo: `다시실행 / Redo / やり直し`
6. Export: `내보내기 / Export / 書き出し`

## 적용 범위 (현재)
1. Home: Quick Start, Service Tracks 카드
2. Studio: 상단 주요 액션, 패널 타이틀, 드로잉 툴
